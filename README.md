# LT Studio — Vitrine Digital

Vitrine editorial multi-página para marca de moda feminina, com curadoria de peças e personal styling. Não é e-commerce transacional — sem carrinho, sem checkout: o objetivo é despertar aspiração na cliente e converter para contato direto via WhatsApp.

## Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router, Turbopack)
- **CMS**: [Sanity](https://www.sanity.io) — gestão de conteúdo (peças, imagens, textos)
- **UI/Animação**: [Tailwind CSS](https://tailwindcss.com), [Framer Motion](https://www.framer.com/motion/), [Lenis](https://lenis.darkroom.engineering/) (scroll suave)
- **Linguagem**: TypeScript

## Comandos

| Comando         | Descrição                          |
| --------------- | ----------------------------------- |
| `npm install`   | Instala as dependências             |
| `npm run dev`   | Sobe o servidor de desenvolvimento (Turbopack) |
| `npm run build` | Gera o build de produção            |
| `npm run start` | Roda o build de produção            |
